# 🚀 快速启动指南

## 为什么 PowerShell 会"停顿"？

PowerShell 在执行某些命令时，会**逐字符显示输入过程**，这看起来像是"停顿"，实际上是正常的显示行为。这不是错误，只是 PowerShell 的特性。

## ✅ 最简单的启动方法

### 方法 1：使用 run.py（推荐）
```bash
python3 run.py
```

### 方法 2：直接运行 app.py
```bash
python3 app.py
```

### 方法 3：使用启动脚本
```bash
./start.sh
```

## 🌐 访问服务

启动成功后，在浏览器中打开：
```
http://localhost:5000
```

## ⚠️ 如果端口被占用

如果看到端口被占用的错误，可以：

1. **查找占用进程**：
   ```bash
   lsof -i :5000
   ```

2. **停止占用进程**（替换 PID 为实际进程号）：
   ```bash
   kill -9 <PID>
   ```

3. **或者修改端口**：编辑 `app.py` 最后一行，将 `port=5000` 改为其他端口（如 `port=5001`）

## 📝 使用步骤

1. **启动服务**：运行 `python3 run.py`
2. **打开浏览器**：访问 `http://localhost:5000`
3. **输入视频 URL**：在页面中输入 YouTube 视频链接
4. **开始下载**：点击"开始下载"按钮
5. **查看进度**：在页面上实时查看下载进度

## 🛑 停止服务

在运行服务的终端中按 `Ctrl + C` 即可停止服务。
